@eris-sharder/core

About

discord-sharder is a pluggable control plane for splitting up shards into groups of "clusters"

Reason

I originally created eris-sharder over a year ago. However, I was pushed by myself, and others, to completely rewrite eris-sharder in order to fix many present issues. Thus, discord-sharder was born.

Installation

To download discord-sharder, run npm install @discord-sharder/core --save
